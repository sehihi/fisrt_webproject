#웹에서 달력 및 일정 관리 구현 (React와 FullCalendar 사용)
npx create-react-app calendar-app
cd calendar-app
npm install @fullcalendar/react @fullcalendar/daygrid @fullcalendar/interaction

React에서 FullCalendar를 사용할 때 월과 요일을 한글로 표시하려면 FullCalendar의 locale 옵션을 설정해주어야 합니다. 
한국어로 설정하기 위해 @fullcalendar/core에서 제공하는 ko 로케일을 사용하면 됩니다.
npm install @fullcalendar/core
3. 웹 애플리케이션의 다른 기능 통합
다양한 기능을 통합하는 웹 애플리케이션을 구축할 때, 달력 외에도 여러 기능을 포함할 수 있습니다. 이를 위해:

라우팅: React Router를 사용하여 다양한 페이지를 관리합니다. 예를 들어, 홈 페이지, 일정 관리 페이지, 사용자 프로필 페이지 등을 설정할 수 있습니다.
npm install react-router-dom
